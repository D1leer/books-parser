# books-parser
# Парсер книг с books.toscrape.com

Собирает названия, цены и наличие книг с сайта-тренажёра. Сохраняет в CSV.

## Как запустить
1. Установить зависимости: `pip install requests beautifulsoup4`
2. Запустить: `python books_parser.py`

## Результат
Файл `books.csv` с колонками: Название, Наличие, Цена
